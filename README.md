Project Description

This project focuses on performing Exploratory Data Analysis (EDA) on a real-world dataset using Python.
The goal is to clean, analyze, visualize, and extract meaningful business insights from raw data.

The project demonstrates strong skills in:

Data Cleaning

Data Transformation

Data Visualization

Trend Analysis

Business Insight Generation

To understand dataset structure and variables

To perform data preprocessing and cleaning

To analyze trends (monthly, yearly, category-wise)

To visualize data using charts and graphs

To generate actionable business insights
